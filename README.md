# EduQuizApp

EduQuizApp is an Android application developed as part of a university project.  
The app helps users learn English vocabulary through lessons and test their knowledge using quizzes.

---

## 📱 Features

- User Registration and Login  
- English Lessons (vocabulary content)  
- Text-to-Speech (pronunciation feature)  
- Quiz system to test knowledge  
- Score and Results screen  
- Settings page with external learning resources  

---

## 🛠️ Technologies Used

- Java  
- Android Studio  
- XML (Layouts)  
- TextToSpeech API  
- Retrofit (for API calls, if used)

---

## 🚀 How to Run the Project

1. Download or clone the repository  
2. Open the project in Android Studio  
3. Let Gradle sync  
4. Run the app on emulator or Android device  

---

## 📦 APK

The APK file is included in this repository:

➡️ `app-debug.apk`

You can install it directly on an Android phone.
